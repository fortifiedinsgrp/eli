# Eli Dashboard

Personal AI assistant dashboard for Marc Singer.

## Features

- **Chat** - Real-time conversation with Eli (OpenClaw integration)
- **Tasks** - Kanban-style task management
- **History** - Timeline of completed work
- **Notes** - Knowledge base and documentation
- **Digests** - Daily activity summaries
- **Documents** - File upload and management

## Tech Stack

- React + Vite
- Tailwind CSS v4
- React Router
- TanStack Query
- Lucide Icons

## Development

```bash
npm install
npm run dev
```

## Deployment

Deployed to Netlify. Push to `main` triggers auto-deploy.

## Backend Integration

- **LifeOS API** - Captain's Log CRM for tasks, notes, companies
- **OpenClaw Gateway** - WebSocket for real-time chat
- **Firebase Storage** - Document uploads (planned)
